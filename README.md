# 🤖 Palinsesto Bot — Guida Setup

## File del progetto
- `bot.py` — codice principale del bot
- `requirements.txt` — librerie Python necessarie
- `Procfile` — istruzioni per Railway

## Deploy su Railway
1. Crea account su railway.app
2. Crea nuovo progetto da GitHub
3. Aggiungi variabile d'ambiente: TELEGRAM_TOKEN = il tuo token
4. Deploy automatico

## Comandi bot
- /start — registrati
- /palinsesto — carica un palinsesto
- /lista — vedi eventi caricati
- /reset — cancella tutti gli eventi
- /status — stato del bot
